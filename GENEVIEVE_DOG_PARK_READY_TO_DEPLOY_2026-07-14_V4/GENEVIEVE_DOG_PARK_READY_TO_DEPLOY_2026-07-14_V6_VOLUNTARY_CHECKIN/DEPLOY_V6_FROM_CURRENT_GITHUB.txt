IMPORTANT CHANGE
Check-in and check-out are now voluntary. Users can continue without checking in.

CURRENT GITHUB METHOD
1. Extract this ZIP.
2. Upload the folder named:
   GENEVIEVE_DOG_PARK_READY_TO_DEPLOY_2026-07-14_V6_VOLUNTARY_CHECKIN
   into the GitHub repository.
3. In Vercel → Project Settings → Build and Deployment → Root Directory:
   use the folder selector and click:
   GENEVIEVE_DOG_PARK_READY_TO_DEPLOY_2026-07-14_V6_VOLUNTARY_CHECKIN
4. Save.
5. Deployments → Redeploy.
6. Turn OFF existing build cache if offered.
7. Open the new Ready deployment.
8. Confirm build 2026.07.14.6.
9. Open Journey → Voluntary Check In / Out.
10. Confirm both buttons appear:
    - Voluntarily check in
    - Continue without checking in

Do not call the feature compulsory unless a competent authority formally adopts and verifies that requirement.
