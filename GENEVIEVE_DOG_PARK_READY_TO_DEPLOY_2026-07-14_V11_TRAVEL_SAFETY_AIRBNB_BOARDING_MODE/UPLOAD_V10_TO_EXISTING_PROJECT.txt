UPLOAD VERSION 10 TO YOUR EXISTING GITHUB PROJECT

1. Extract this ZIP.
2. Open the extracted folder. You should immediately see index.html, app.js, styles.css, service-worker.js, manifest.webmanifest, assets, legal and docs.
3. In the existing GitHub repository, choose Add file → Upload files.
4. Drag ALL extracted files and folders onto the upload page together.
5. Let GitHub replace matching V9 files and add the new V10 files.
6. Commit the changes.
7. In Vercel, redeploy the latest commit if it does not begin automatically.
8. Open the website in a private window and confirm the footer says Dog Park build 2026.07.14.10.
9. On iPhone, remove the old home-screen shortcut before adding the website again so iOS cannot keep the old paw icon.
10. In the app, More → Refresh latest app version clears old GENEVIEVE website caches without deleting saved dog records.

Do not choose a Root Directory or Build Command for this static package. index.html is at the repository root.
